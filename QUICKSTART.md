# ⚡ VISION-GUARD Quick Start

Get healthy work habits in 2 minutes!

## 🚀 Installation

```bash
# Clone repository
git clone https://github.com/OrbitScript/vision-guard.git
cd vision-guard

# Install dependencies
pip install -r requirements.txt
```

## 🎬 First Session

### 1. Start Monitoring
```bash
python vision_guard.py
```

### 2. Position Yourself
- Sit at your desk
- Face your webcam
- Maintain good posture
- Start working!

### 3. See Real-Time Feedback
Watch the overlay on your screen:
- ✅ **Green** - Good posture!
- ⚠️ **Yellow** - Minor issues
- 🔴 **Red** - Fix your posture!

### 4. View Statistics
Press `S` to see:
- Focus time
- Break time
- Posture warnings
- Productivity score

## ⌨️ Hotkeys

- `Q` - Quit and save session
- `S` - Toggle stats overlay
- `P` - Pause monitoring

## 📊 View Analytics

After a few sessions:
```bash
python analytics.py
```

See beautiful charts of:
- Focus time trends
- Productivity scores
- Warning distribution
- Time balance

## 💡 Tips for Day 1

### Camera Setup:
✅ Position at eye level  
✅ Good lighting (face clearly visible)  
✅ Arm's length distance  
✅ Webcam permission granted  

### Posture Basics:
✅ Head level, not tilted  
✅ Eyes aligned with top 1/3 of screen  
✅ Sit back in chair  
✅ Feet flat on floor  

### Expected Alerts:
- "Too Close" - Move back a bit
- "Head Tilted" - Straighten up
- "Looking Away" - Refocus on screen

## 🎯 Your First Day Goal

1. Start VISION-GUARD

4. Review your stats
5. See your productivity score!

**Target Score: 70+** for first session

## ❓ Common Questions

**Q: Camera not working?**  
A: Check webcam permissions, close other apps using camera

**Q: No face detected?**  
A: Ensure good lighting, position face toward camera

**Q: Too many warnings?**  
A: Adjust your setup, improve posture, sit further back

**Q: How to improve score?**  
A: Maintain good posture, take regular breaks, stay focused
